Title: Head tracking utilizing a manufactured 3-D world 

The goal of our project was to create a tracking software in Python that detects a user’s head movement using a live-video screen in relation to a manufactured 3-D world. 
To do this, we first calibrated the user's head movement, in order for the program to recognize when the user is turning their head. 
This was done by hightling the dots on the face using minumin and maximum cofidence dections to find the main landmarks on the face (Ex: Eye, Noise, Mouth) 
Once calibrated, a three-dimensional map appears that we built. This map has a road with two intersecting roads, having uniquely built building throughout 
Users can move around in the map by turning their head in the directions of left, right, up, down, front, and back. 

We utilized Python programming software in writing this code 

Additionally to help us accomlish this task, we used the LLM Cursor to assist in our head tracking coding project. We used this LLM by asking it for suggestions on how to best implement the coding steps in our plan. 
From there we used these suggestions to have the LLM explain new programing skills and features of python that we have not yet learned in class such as the Panda 3-D and MediaPipe functions. Using this we impemented the code by learning its features and expanding / editing the LLM sugggstions. 
Using the LLM not only taught us how to use specific packages and functions in python, but also taught us how to probe LLM's to be benificial in coding languages. 

Our code that includes the headtracking technology / calibration is under the file name "track.py" 
The code that includes our 3-D world is under the file name "test.py" 
Finnaly our code that combines these files in order for the head tracking to move the user in the 3-D world is under the fil name "moving.py"


We hope you enjoy moving around in our 3D city! (Press C to calibrate your head ... then to move in the world look either left, right, up down, fowards, or backwards to move in that direction in the world)
